# Projetos de Desenvolvimento Web

Este repositório reúne projetos desenvolvidos durante o meu primeiro semestre da faculdade utilizando HTML, CSS e JavaScript, com o objetivo de praticar os fundamentos do desenvolvimento web.

## Projetos

🧮 Calculadora

Calculadora simples capaz de realizar operações matemáticas básicas, desenvolvida para praticar lógica de programação.

🕰️ Relógio Analógico

Relógio com ponteiros que exibem a hora atual em tempo real, utilizando JavaScript para atualizar os movimentos de horas, minutos e segundos.

✅ Lista de Tarefas

Aplicação que permite adicionar, concluir e remover tarefas, auxiliando na prática de eventos e manipulação de elementos da página.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
